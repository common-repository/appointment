== Description ==
 
**Appointment booking** gives you the ability to Appointment booking for next 7 days, Where admin can set time According to requirements.  

Features of **Appointment booking** plugin:

1. Admin Can set time slots according to need.
2. Admin can set different timing for meeting or appointment. 
3. Admin can also set colours for the calendar pop-up and text etc.
4. Admin can set appointment success/failed message message for customer 
5. Admin can set customer information fields , which is required or not.
 
 
== Installation ==

#### From within WordPress

1. Visit 'Plugins > Add New'
2. Search for 'Appointment'
3. Activate 'Appointment' for WordPress from your 'Plugins' menu from WordPress.
4. Visit the  'Appointment' menu.

#### Manually

1. Upload the appointment/ folder to the /wp-content/plugins/ directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress. 
3. Visit the  'Appointment' menu.
